# CAN-Message-Simulator
C#（WinForms）で開発した、車載CANメッセージの送信を模擬するツールです。  
複数のCAN IDをランダムに切り替えながら、速度・回転数・温度などの車載データを生成し、UI上でリアルタイムに表示します。  
エラー注入機能やCSV形式でのログ保存にも対応しており、車載ECU開発やCAN通信テストの基礎学習に利用できます。

## 主な機能
- 複数CAN IDのランダム切り替え（0x100 / 0x120 / 0x130 / 0x200）
- 車載データの自動生成  
  - 速度（0〜180 km/h）  
  - 回転数（0〜6000 rpm）  
  - 温度（-20〜120°C）
- 200ms周期でのリアルタイム更新
- エラー注入（0xFF埋めCANデータ）
- CSV形式でのログ保存
- シンプルで扱いやすいWinForms UI

## このツールの目的
車載ソフトウェア開発で扱われるCAN通信の基礎を理解するために作成したツールです。  
実際の車載ログビューアやCAN解析ツールを意識して設計しており、  
CANメッセージの周期処理、データ生成、UI更新、ログ管理などの基本的な仕組みを学べます。

## 使用技術

- Windows Forms
- Timerによる周期処理
- CSVログ出力
